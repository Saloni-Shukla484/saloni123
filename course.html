Web Development

What is web development? Web development typically refers to the coding and programming side of website production. When you learn web development, you might start out writing a simple page of HTML text and build up to creating more complex, feature-rich applications designed to be accessed from various Internet-connected devices. Examples of feature-rich web development include e-commerce websites, content management systems (CMS), and social networks. Common programming languages and software web developers may use include Hypertext Markup Language (HTML), Cascading Style Sheets (CSS), JavaScript, PHP, Drupal, and MySQL..
